# ai-sql-gen 🤖

AI-powered SQL query generator from natural language.

## Features ✨
- **AI-Powered**: Leverages advanced LLMs for intelligent analysis and generation.
- **Fast & Efficient**: Designed for quick terminal-based workflows.
- **Easy to Use**: Simple CLI interface with clear output.

## Installation 🚀
1. Clone the repository:
   ```bash
   git clone https://github.com/rawqubit/ai-sql-gen.git
   cd ai-sql-gen
   ```
2. Install dependencies:
   ```bash
   pip install openai rich click
   ```
3. Set up your OpenAI API key:
   ```bash
   export OPENAI_API_KEY='your-api-key-here'
   ```

## Usage 🛠️
```bash
python main.py --help
```
